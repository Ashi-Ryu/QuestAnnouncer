# QuestAnnouncer
Announces quest objective progress or completion to party chat.
"Commands:"
    /qa filter accept|progress|complete|turnin - Toggle announcement filters")
    /qa prefix short|long - Change prefix style")
    /qa debug - Toggle debug output")
    /qa test - Send test announcement")
    /qa status - Show current settings")

GUI options - Interface > AddOns > QuestAnnouncer
